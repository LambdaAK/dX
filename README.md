# dX · Applied Math & ML Sandbox

Interactive labs for applied mathematics and machine learning in the browser. Each lab lets you explore concepts with live simulations and visualizations.

**Topics:** Linear algebra (factorizations, eigenvalues, solve Ax=b) · Probability (LLN, CLT, concentration inequalities) · Stochastic processes (SDEs, Markov chains, CTMCs) · Regression & classification (linear, logistic, KNN, trees, bagging, boosting, perceptron, SVM) · Clustering (K-Means, DBSCAN) · Dimensionality (PCA) · Reinforcement learning (bandits, MDPs) · Optimization (simplex, QP, gradient descent) · Physics (pendulum, heat equation 1D/2D/3D)

**Live:** https://dxlab.netlify.app/

**Local:** `npm install && npm run dev` → http://localhost:5173
